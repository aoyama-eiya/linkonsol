# LinkOnSol

LinkOnSol is a privacy-first, serverless Web3 "Link in Bio" portfolio builder on Solana.

## Libraries & Acknowledgements

This project makes use of the following open-source libraries:

- **Next.js** - React framework for production.
- **Tailwind CSS** - Utility-first CSS framework.
- **@solana/wallet-adapter** - Comprehensive set of libraries for Solana wallets.
- **Framer Motion** - Production-ready motion library for React.
- **Lucide React** - Beautiful & consistent icon toolkit.
- **React Icons** - Include popular icons in your React projects.
- **clsx** - A tiny utility for constructing className strings conditionally.
